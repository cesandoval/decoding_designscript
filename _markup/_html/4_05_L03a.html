4.05.L03a Given a Surface and a swap bool, divide the Surface evenly along its isoparms to develop toolpaths for a CNC mill. required: surf (Surface) A Surface to create toolpaths from. swap (bool) Whether or not to swap the direction of the toolpaths. results: lns([PLine]) PLines representing the cutpaths.
Method to generate a surface.

import("ProtoGeometry.dll");

def func(u, v)
 
{
 
length = 4.0;
 
rad = (Math.Cos(Math.RadiansToDegrees(MapTo(0, 1, v, 0, Math.PI * 2 - 1.5))) + 1.75) / 2;
 
u = MapTo(0, 1, u, 0.1, 0.9);
 
 
 
x = rad * Math.Cos(Math.RadiansToDegrees(u * Math.PI));
 
y = MapTo(0, 1, v, 0, length);
 
z = rad * Math.Sin(Math.RadiansToDegrees(u * Math.PI));
 
return = Point.ByCoordinates(x, y, z);
 
}
